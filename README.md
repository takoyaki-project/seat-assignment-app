# 🪑 席替えアプリ

**出席番号だけで使える、名前不要の条件付き席替えWebアプリ。**

<img width="1586" height="992" alt="ogp-sekigae" src="https://github.com/user-attachments/assets/6a477e07-b5ca-4b0b-b795-bdc592353e38" />

👉 **今すぐ使う： https://takoyaki-project.github.io/seat-assignment-app/**

インストール不要・ログイン不要・無料。ブラウザで開けばすぐ使えます。

---

## こんなときに

- 席替えのたびに、くじ引きの紙を作るのが地味に大変
- 「あの子とあの子は隣にしない」などの配慮を、公平さを保ちながら反映したい
- 視力などの理由で前の方に座らせたい子がいる
- 決まった座席表を、そのまま印刷して掲示したい

## 🔒 いちばん大事にしたこと：個人情報を扱わない

このアプリは**児童・生徒の名前を一切入力しません**。使うのは出席番号だけ。

- サーバーへのデータ送信なし（すべて端末のブラウザ内で完結）
- アカウント登録・ログインなし
- クラス設定の保存も、お使いの端末のブラウザ内（localStorage）のみ

学校で使うツールだからこそ、「そもそも預からない」設計にしました。

| 条件 | できること |
|---|---|
| 🙅 隣席NG | 「1番と2番は隣にしない」を何組でも設定 |
| 👀 前列指定 | 「3番は前から2行目まで」など、前方の席を保証 |
| 📌 指定席 | 特定の席に固定したい生徒を指定 |
| 👥 男女配置ルール | 交互配置や、列ごとの男女指定に対応 |

### ステップ3：席替え実行！

条件をすべて満たす配置を自動で探します。結果の座席表はそのまま **🖨️ 印刷** できます。

## その他の機能

- 💾 **クラス情報の保存** — 人数や条件をブラウザに保存。次回の席替えは開くだけ
- 📱 **スマホ対応** — 職員室のPCでも、教室のタブレットでも
- 🖨️ **印刷対応** — 座席表をそのまま掲示物に

## よくある質問

**Q. データはどこに保存されますか？**
A. お使いの端末のブラウザ内だけです。外部への送信は一切ありません。ブラウザの履歴・サイトデータを消去すると保存内容も消えます。

**Q. 料金は？**
A. 無料です。登録も不要です。

**Q. 何人まで対応していますか？**
A. 一般的なクラスサイズ（〜40人程度）を想定しています。
