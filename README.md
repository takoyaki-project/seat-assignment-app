# 席替えアプリ

出席番号だけで使える、名前不要の条件付き席替えWebアプリ。

## 特徴

- ダウンロード不要
- ログイン不要
- 個人情報の取得なし
- スマホ対応
- 隣席NG設定
- 前列指定
- 指定席設定
- 男女配置ルール
